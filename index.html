神奇的css！竟然可以这样玩转表格

这是在对一个博客模板进行移动端适配时遇到的一个场景。html结构如下：

<table width=100% class="Framework" cellspacing="0" cellpadding="0">
    <tr>.....</tr>
    <tr>
        <td class="LeftCell">..</td>                    <td class="MainCell" width="100%">..</td>   
    </tr>
    <tr>.....</tr>
</table>

要解决的问题是如何在不修改任何html代码的情况下，仅仅通过css实现下面的效果：

1）改变它们的显示顺序，.MainCell显示在前，.LeftCell显示在后（与html结构中的顺序相反）。

2）让它们分行显示。 .MainCell与.LeftCell是表格的单元格，它们默认是以列的方式排列在一行显示，如果解决了第1个问题，那就是.MainCell在左，.LeftCell在右；让它们分行方式显示就是.MainCell显示在上面，.LeftCell显示在下面。

要解决这2个问题，首先要解决1个选择器的问题 —— 如何在没有id与class的情况下，让添加的样式只针对表格（table）的第2行（tr）生效。

css3中有个伪类选择器可解决这个问题，它就是 :nth-child() （参考资料：CSS3 :nth-child()伪类选择器），于是就有了下面的选择器：

.Framework tr:nth-child(2){
}

接下来解决上面的第1个问题 —— 改变元素的显示顺序。

css3中引人了一个新的布局模型 —— flex （弹性盒模型，flexible box的缩写，参考资料：深入理解CSS弹性盒模型flex）。使用flex布局，可以通过order控制显示顺序（order值小的显示在前面）。

于是，通过下面的css代码将.LeftCell与.MainCell改为flex布局：

.Framework tr:nth-child(2){
    display: flex;
}

要将.MainCell显示在.LeftCell之前，只需给.LeftCell的order设置一个大于0的值（默认是0，值越大显示越后）即可：

.LeftCell{
    order:1;
}

第1个问题解决。

紧接着解决第2个问题，让.MainCell与.LeftCell分行显示，也就是显示.MainCell之后，换一行显示.LeftCell。

css3中强大的flex布局继续发挥作用，通过flex-wrap可以控制flex布局中元素的换行方式，对于我们要解决的问题，使用”wrap”即可：

.Framework tr:nth-child(2){
    flex-wrap: wrap;
}

这篇文章的亮点在于flex，想了解更多 请登录 http://www.cnblogs.com/xiaohuochai/p/5323146.html

超实用的知识点 

CSS样式
<style>
    .div1{
        width: 100px;
        height: 100px;
        background: red;
        position: absolute;
        top: 0px;
        left: 0px;
    }
    .box{
        border: 1px solid #000;
        position: absolute;
    }
</style>

div布局

<div class="div1">

</div>

JS模块处理

<script>
定义变量
    var div1=document.querySelector(".div1");
    var disX=0;
    var disY=0;
    var timer=null;


通过移动的最后两个点来找速度和方向
    var prevX=0;
    var prevY=0;
    var iSpeedX=0;
    var iSpeedY=0;
鼠标按下时    
    div1.onmousedown=function(ev){
        var ev=ev||window.event;
        disX=ev.clientX-div1.offsetLeft;
        console.log(ev.clientX,div1.offsetLeft)
        disY=ev.clientY-div1.offsetTop;

        prevX=ev.clientX;
        prevY=ev.clientY
        clearInterval(timer)
        document.onmousemove=function(ev){
        速度就是两个点得差值
            var ev=ev||window.event;
            div1.style.left=ev.clientX-disX+"px";
            div1.style.top=ev.clientY-disY+"px";
            iSpeedX=ev.clientX-prevX;
            iSpeedY=ev.clientY-prevY;
        一直保持最后两个点得差值
            prevX=ev.clientX;
            prevY=ev.clientY;
        }
        document.onmouseup=function(){
            document.onmousemove=null;
            document.onmouseup=null;
            startMove();
        }
    让小球移动的函数
    function startMove(){
        clearInterval(timer)
        timer=setInterval(function(){
        // 自由落体
        iSpeedY+=3;
        var L=div1.offsetLeft+iSpeedX;
        var T=div1.offsetTop+iSpeedY;
        if(L<0){
            L=0;
            // 控制方向
            iSpeedX*=-1;
            // 摩擦系数
            iSpeedX*=0.75;
        }else if(L>document.documentElement.clientWidth-div1.offsetWidth){
            L=document.documentElement.clientWidth-div1.offsetWidth;
            iSpeedX*=-1;
            iSpeedY*=0.75;
            iSpeedX*=0.75;
        }
        if(T<0){
            T=0;
            iSpeedY*=-1;
        }else if(T>document.documentElement.clientHeight-div1.offsetHeight){
            T=document.documentElement.clientHeight-div1.offsetHeight;
            iSpeedY*=-1
        }
        div1.style.left=L+"px";
        div1.style.top=T+"px";
    },30)
}
</script>

上述代码的实现是对于一些简单的js逻辑上的处理 比如对clientX,clientY,以及对dom的一些使用，我个人认为这里面最值得学习的是，特别是针对新手，很重要，这里面的碰撞检测是大多数游戏中或是一些JS实例中最容易体现，希望采纳
