tab选项卡

前段时间有个网友问我京东上的那个动态效果图是怎么做出来的,其实这种效果很常见的在很多网站上都有看到，所以我再博客中做了这么一篇简易直接的效果

Js代码如下：

var btn=document.querySelectorAll("ol li");
var pic=document.querySelectorAll("ul li");
var left=document.querySelector(".left");
var right=document.querySelector(".right");
var num=0;
for(var i=0;i<btn.length;i++){
    btn[i].index=i;
btn[i].onmouseover=function(){
    for(var i=0;i<btn.length;i++){
        btn[i].className="";
        pic[i].className="";
    }
    btn[this.index].className="active";
    pic[this.index].className="active";
}
}

left.onclick=function(){
    num--;
    if(num<0){
        num=btn.length-1;
}
for(var i=0;i<btn.length;i++){
    btn[i].className="";
    pic[i].className="";
}
btn[num].className="active";
pic[num].className="active";
}

right.onclick=function(){
    num++;
    if(num>btn.length-1){
        num=0;
}
for(var i=0;i<btn.length;i++){
    btn[i].className="";
    pic[i].className="";
}
    btn[num].className="active";
    pic[num].className="active";
}

js这块仔细想想起始逻辑还好只要你布局没问题，这块稍微注意一下应给就能写出来

CSS代码块

*{
        padding: 0;
        margin:0;
    }
    ol,ul{
        list-style: none;
    }
    .tab{
        width: 500px;
        height: 400px;
        margin: 50px auto;
        position: relative;
        border: 1px solid red;
    }
    .btn{
        width: 50px;
        height: 70px;
        background: black;
        position:absolute;
        color: white;
        opacity: 0.5;
        line-height: 70px;
        text-align: center; 
        top: 165px;
        z-index: 100;
        font-size: 30px;
    }
    .left{
        left:0px;
    }
    .right{
        right:0px;
    }
    .tab ul{
        width: 500px;
        height: 400px;
        overflow: hidden;
        position: absolute;
    }
    .tab ul li{
        position: absolute;
        width: 500px;
        height: 400px;
        opacity: 0;
    }
    .tab ul .active{
        opacity: 1;
    }
    .tab ol{
        position: absolute;
        width: 200px;
        height: 30px;
        bottom: 10px;
        background: black;
        left: 50%;
        margin-left: -100px;
    }
    .tab ol li{
        width: 30px;
        height: 30px;
        float: left;
        margin-left: 16px;
        border-radius: 50%;
        background: cyan;
    }
    .tab ol .active{
        background: gray;
    }

页面布局是很重要的，他决定你在写JS时候的简单或复杂

<div class="tab">
    <div class="btn left">&lt</div>
    <div class="btn right">&gt</div>
    <ul>
        <li class="active"><img src="image1.jpg"></li>
        <li><img src="image2.jpg"></li>
        <li><img src="image3.jpg"></li>
        <li><img src="image4.jpg"></li>
    </ul>
    <ol>
        <li class="active"></li>
        <li></li>
        <li></li>
        <li></li>
    </ol>
</div>

上面这些代码就是一个完整的选项卡 这也能算是个模板 在其他地方一样使用；tab选项卡对于初学者有很好的体验
