php

##php基础学习

echo phpinfo() 查看php版本信息

php5.5以下用的是mysql来连接（面向过程） 5.5以上用的时mysqli来连接(面向过程/面向对象)

echo只能打印数字或字符串，打印数组print_r()

php三种注释

1.第一种注释 /* */
2.第二种注释  //  单行注释
3.多行注释  /***  换行     ****/

变量

$a="你好"；
echo $a;

$b="hello";
$hello="hello world";
echo $$b // hello world

变量解析

$c="你好"；
$res="张三"；
$res1=$res.$c 
echo $res1  //张三你好
echo "张三$c"  //双引号里面会解析变量，单引号不能
echo '张三'.$c.'hello';
echo  "张三{$c}hello";

类型转换

$d=1;
$e="2";
echo ($d+$e);  //自动转换
echo gettype((int)$e)  //类型强制转换
echo gettype(($d+$e))

字符串的另一个定义方式 ：定界符 一般定义长字符串 注意：结尾要顶头写 前后都不能有空格；结束后面也不能写东西

$str=<<<AAA
fagsggfdh
AAA;
echo $str;

数组

$arr=array("张三","李四","小儿");  //索引数组
print_r($arr);

$arr3=array("name1"=>"张三","name2"=>"李四","name3"=>"小儿"); // 定义关联数据
print_r($arr3);
unset() //删除数组元素
is_array() //判断是否在数组内
is_type（）//判断是什么类型

全局变量

$_COOKIE $GLOBALS  $SESSION $_FILES $_SERVER

定义常量 ：名称一般大写，只能被定义一次 ,第三个参数默认是false，区分大小写；true，就是不区分大小写

define("NAME","张三")

类

class Test{
    function fun(){
        echo __CLASS__;  //显示类名
        echo __METHOD__; // 显示类名下的方法名
    }
}
$test = new Test();
$test->fun();
